# Data Immersion & Wrangling Project

## Objective
The objective of this project is to acquire, analyze, clean, and transform raw data into an analysis-ready dataset. This project demonstrates data profiling, quality assessment, data cleaning, feature engineering, and documentation using Python and Pandas.

## Dataset Overview
The dataset contains customer transaction information including customer details, purchase amounts, and purchase dates.

## Tasks Performed

### 1. Data Access & Familiarization
- Loaded dataset using Pandas
- Studied column names and data types
- Created a Data Dictionary

### 2. Data Quality Assessment
- Identified missing values
- Checked duplicate records
- Detected inconsistent formatting
- Examined outliers

### 3. Data Cleaning & Transformation
- Handled missing values
- Removed duplicates
- Standardized date formats
- Corrected text formatting
- Created new features such as Customer Age
- Prepared final analysis-ready dataset

## Tools Used
- Python
- Pandas
- NumPy
- VS Code / Jupyter Notebook

## Deliverables
- Data Dictionary
- Data Cleaning Script
- Cleaned Dataset
